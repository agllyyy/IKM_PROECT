*Индивидуальный проект по дисциплине «Программирование на Python»*

# Описание проекта 
Разработать и реализовать программу "Учёт сотрудников", которая считывает исходную информацию и позволяет создавать отчёты:
- полный список всех сотрудников
- список сотрудников заданного отдела
- список сотрудников с окладом выше среднего
  
# Запуск и установка

### Требования:
- версия Python 3.10 или выше
- Файл `working.csv` в той же папке, что и программа  
> Программа использует только встроенные модули Python - дополнительные библиотеки не требуются.

### Установка:
1.  Скачайте файлы проекта `progrm.py` и  `working.csv`
2.  Убедитесь, что у вас установлен Python 3.10+
3.  Поместите файл данных `working.csv` в ту же папку, что и `progrm.py`

# Как использовать:

### Главное меню:
1. Показать всех учеников
2. Показать именинников по сезону
3. Показать учеников одного класса
4. Выход
